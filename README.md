# 🌿 灵枢安 - 家庭健康管家

家庭健康的守护者

## 功能特点

- 📊 健康数据管理（血压、血糖、心率、体重）
- 💊 用药提醒和疫苗提醒
- 🥗 AI智能饮食建议
- 💬 AI健康咨询
- 👨‍👩‍👧‍👦 家庭成员管理
- 🌙 夜间模式
- 🎤 语音输入

## 安装使用

1. 下载安装包（待上传）
2. 双击运行安装
3. 打开浏览器访问 http://127.0.0.1:5000

## 技术栈

- 后端：Flask + SQLAlchemy
- 前端：HTML/CSS/JavaScript
- AI：DeepSeek API
- RAG：Sentence-Transformers + ChromaDB
